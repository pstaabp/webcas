Feasible-Set Plotter

To use this tool you must be using a modern browser. See the list of supported browsers.

To begin, enter all linear contraints. Use <= and >= for less than (greater than) and equal to. Use x and y as the two variables and put each inequality in the form ax+by<=c or ax+by>=c. Then click on the Start button below.
